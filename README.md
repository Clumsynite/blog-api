# Blog API

## Introduction

Trying out REST Api

## Endpoints

### Authentication

#### User Login
  
- **POST** *`/auth/login`*
  
#### User Logout

- **POST** *`/auth/logout`*

### User Related

#### User Signup

- **POST** *`/user/signup`*

#### Update User

- **PUT** *`/user/:id/update`*

#### View a User's Profile

- **GET** *`/user/:id/view`*

#### View your profile

- **GET** *`/user/me`*

#### View a user's blogs

- **GET** *`/user/:id/blogs`*

#### View your Draft Posts

- **GET** *`/user/drafts`*

#### View a user's comments

- **GET** *`/user/:id/comments`*
